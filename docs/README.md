# DoomScrollers Mini-ITC Documentation

## Overview
This repository contains the MissaTech breach analysis pipeline used by Team DoomScrollers for the Mini-ITC case study. The project ingests breach data from the CSV dataset, builds a local SQLite database, runs analytics and ML models, and produces visualizations, executive summaries, and a monitoring dashboard. The primary entrypoint is `main.py`, which orchestrates the end-to-end workflow and offers interactive options for generating presentations or launching the dashboard.【F:main.py†L1-L409】

## Quickstart
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the full analysis pipeline:
   ```bash
   python main.py
   ```
   - This loads the CSV dataset into SQLite, runs analysis functions, generates charts, executes ML models, and prompts for optional presentation/dashboard steps.【F:main.py†L1-L409】
